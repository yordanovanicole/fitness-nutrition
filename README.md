# Fitness & Nutrition Website
Responsive fitness and healthy lifestyle website developed using HTML, CSS and JavaScript.

## Features
- Calorie calculator
- Nutrition section
- Workout section
- Responsive design
- JavaScript form validation
- Dynamic calorie calculation
- Embedded YouTube fitness video using iframe

## Technologies
- HTML5
- CSS3
- JavaScript
